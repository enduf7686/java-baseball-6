# 📝 기능 명세서

### 번호(Number)

- [x] 리스트의 길이가 3인지 체크
- [x] 숫자가 1부터 9 사이인지 체크
- [x] 중복된 숫자가 있는지 체크

### 번호 생성기

- [x] 랜덤 번호 생성

### 컴퓨터

- [x] 번호 생성기로 부터 받은 번호 저장

### 번호 비교

- [x] 컴퓨터의 숫자와 사용자가 입력한 숫자를 비교해서 결과 출력

### 결과

- [x] 볼, 스트라이크의 갯수 저장

### 📥 입력(Input)

- [ ] 입력 받은 문자열이 숫자로 이루어져 있는지 체크

### 📤 출력(Output)

- [ ] 콘솔을 이용해 사용자에게 결과를 출력해준다.
